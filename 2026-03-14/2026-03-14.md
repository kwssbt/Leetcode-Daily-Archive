## DFS

`dfs` 生成所有长度为 $n$ 的开心字符串

返回第 $k$ 个,不足 $k$ 个则返回空串

依次尝试填充 $a,b,c$ 自动满足升序

遇到邻位相同时跳过


---

## 代码实现 (C++)
```cpp
class Solution {
public:
    string getHappyString(int n, int k) {
        vector<string>ans;
        string path="";
        function<void()>dfs=[&]()->void{
            if(path.size()==n){
                ans.push_back(path);
                return;
            }
            for(char i='a';i<='c';++i){
                if(path.size()&&path.back()==i)continue;
                path.push_back(i);
                dfs();
                path.pop_back();
            }
        };
        dfs();
        return ans.size()<k?"":ans[k-1];
    }
};
```

- 时间复杂度: $O(3^N)$
- 空间复杂度: $O(N\times2^N )$

空间复杂度为 $N \times 长度为 N 的开心字符串的数量 $

第一个位置可以放 3 个元素

之后每个位置只能放 2 个元素

所以长度为 $n$ 的开心字符串的数量为 $3 \times 2^{n-1}$

---

## 优化

1. 长度为 $n$ 的开心字符串的数量为 $3 \times 2^{n-1}$，由此提前判断是否需要返回空串
2. 答案只需要第 $k$ 个，只需记数，找到第 k 个就结束递归

---

## 代码实现 (C++)
```cpp
class Solution {
public:
    string getHappyString(int n, int k) {
        if(k>3*(1<<(n-1))){
            return "";
        }
        string ans;
        string path="";
        int cnt=0;
        bool found=0;
        function<void()>dfs=[&]()->void{
            if(found){
                return;
            }
            if(path.size()==n){
                if(++cnt==k){
                    found=1;
                    ans=path;
                }
                return;
            }
            for(char i='a';i<='c';++i){
                if(path.size()&&path.back()==i)continue;
                path.push_back(i);
                dfs();
                path.pop_back();
            }
        };
        dfs();
        return ans;
    }
};
```

- 时间复杂度: $O(3^N)$
- 空间复杂度: $O(N)$
