## 枚举

简化问题

现有 h = [2,1,4,5,6]，表示 5 根柱子的高度

可以按任意顺序排列，求可以构成的最大矩形

从大到小排序 [6,5,4,2,1]

枚举可能的矩形

从左到右枚举矩形的右边界

考虑将第 i 根柱子作为右边界，高h[i]

左侧都不低于h[i]，一定可以构成 高h[i]，长i+1，的矩形

右侧都不大于h[i]。若小于，无法并入以 h[i] 为高的矩形。若等于，以它为右边界的矩形会覆盖当前的矩形。

枚举底边，从而枚举出所有可能的矩形

## 代码实现 (C++)
```cpp
class Solution {
public:
    int largestSubmatrix(vector<vector<int>>& matrix) {
        int n=matrix.size();
        int m=matrix[0].size();

        // h[i][j]：以第i行为底边，第j根柱子的高度
        vector h(n,vector<int>(m));
        
        for(int i=0;i<n;++i){
            for(int j=0;j<m;++j)
            if(i==0){
                h[i][j]=matrix[i][j];
            }
            else{
                if(matrix[i][j]){
                    h[i][j]=h[i-1][j]+1;
                }
                else{
                    h[i][j]=0;
                }
            }
        }

        int ans=0;
        
        for(int i=0;i<n;++i){

            auto&cur=h[i];
            sort(cur.rbegin(),cur.rend());
            
            int mx_s=0;
            for(int j=0;j<m;++j){
                mx_s=max(mx_s,cur[j]*(j+1));
            }
            ans=max(ans,mx_s);
        }
        return ans;
    }
};
```

- 时间复杂度: $O(NM\log M)$
- 空间复杂度: $O(NM)$
