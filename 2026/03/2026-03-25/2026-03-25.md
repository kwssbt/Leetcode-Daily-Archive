## 枚举

先求总和 $SUM$

$SUM$ 为奇提前返回 0

分别横着，竖着求和。换行（列）时判断是否达到 $\frac{SUM}{2}$

---

## 代码实现 (C++)
```cpp
class Solution {
public:
    bool canPartitionGrid(vector<vector<int>>& grid) {
        int n=grid.size();
        int m=grid[0].size();
        long long sum=0;
        for(int i=0;i<n;++i){
            for(int j=0;j<m;++j){
                sum+=grid[i][j];
            }
        }
        if(sum&1)return 0;
        long long target=sum>>1;
        long long cur=0;
        for(int i=0;i<n;++i){
            for(int j=0;j<m;++j){
                cur+=grid[i][j];
            }
            if(cur==target){
                return 1;
            }
        }
        cur=0;
        for(int j=0;j<m;++j){
            for(int i=0;i<n;++i){
                cur+=grid[i][j];
            }
            if(cur==target){
                return 1;
            }
        }
        return 0;
    }
};
```

- 时间复杂度: $O(NM)$
- 空间复杂度: $O(1)$
