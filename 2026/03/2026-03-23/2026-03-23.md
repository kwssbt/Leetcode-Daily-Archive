# [1594. 矩阵的最大非负积](https://leetcode.cn/problems/maximum-non-negative-product-in-a-matrix/)

## 动态规划

由于负负得正，我们需要同时维护最大乘积和最小乘积

`Max[i][j]`: 从 [0][0] 到 [i][j] 的最大乘积

`Min[i][j]`: 从 [0][0] 到 [i][j] 的最小乘积

采用 "pull DP"

对于 [0][0]: 乘积就是自身

对于 [i][0] (i>1) : 只能来自 [i-1][0]

对于 [0][j] (j>1) : 只能来自 [0][j-1]

对于 [i][j] (i,j>1) : 可以来自 [i-1][j] 或 [i][j-1]

---

## 代码实现 (C++)
```cpp
class Solution {
    typedef long long ll;
    static const int MOD=1e9+7;
public:
    int maxProductPath(vector<vector<int>>& grid) {
        int n=grid.size();
        int m=grid[0].size();
        vector Max(n,vector<ll>(m));
        vector Min(n,vector<ll>(m));
        for(int i=0;i<n;++i){
            for(int j=0;j<m;++j){
                int v=grid[i][j];
                if(i==0&&j==0){
                    Max[i][j]=Min[i][j]=v;
                }
                else if(i==0){
                    Max[i][j]=Min[i][j]=Max[i][j-1]*v;
                }
                else if(j==0){
                    Max[i][j]=Min[i][j]=Max[i-1][j]*v;
                }
                else{
                    ll a=Max[i-1][j]*v;
                    ll b=Min[i-1][j]*v;
                    ll c=Max[i][j-1]*v;
                    ll d=Min[i][j-1]*v;
                    Max[i][j]=max({a,b,c,d});
                    Min[i][j]=min({a,b,c,d});
                }
            }
        }
        ll ans=Max[n-1][m-1];
        return ans<0?-1:ans%MOD;
    }
};
```

- 时间复杂度: $O(NM)$
- 空间复杂度: $O(NM)$
