## 除法同余

设所有数的积为 $MUL$

$$ p[i][j] = \frac{MUL}{grid[i][j]} $$ 

除法取余需要计算逆元

费马小定理要求 $MOD$ 为质数

拓展欧几里得需要 $X$ 与 $MOD$ 互质

本题 $MOD=12345$，均不满足

所以无法使用除法。

## 前后缀

将【$n\times m$】二维矩阵压成一维：

$key=i*m+j$

$i=\frac{key}{m}$

$j=key-i\times m$

$pre[key]$ 表示 $grid[i][j]$前面的所有数字累积，不含$grid[i][j]$

$suf[key]$ 表示 $grid[i][j]$和后面的所有数字累积，含$grid[i][j]$

$p[i][j]=pre[key]\times suf[key+1]$

---

## 代码实现 (C++)
```cpp
class Solution {
    const int MOD=12345;
public:
    vector<vector<int>> constructProductMatrix(vector<vector<int>>& grid) {
        int n=grid.size();
        int m=grid[0].size();
        vector<long long>pre(n*m+1);
        pre[0]=1;
        for(int i=0;i<n;++i){
            for(int j=0;j<m;++j){
                pre[i*m+j+1]=pre[i*m+j]*grid[i][j]%MOD;
            }
        }
        vector<long long>suf(n*m+1);
        suf[n*m]=1;
        for(int i=n-1;i>=0;--i){
            for(int j=m-1;j>=0;--j){
                suf[i*m+j]=suf[i*m+j+1]*grid[i][j]%MOD;
            }
        }
        vector p(n,vector<int>(m));
        for(int i=0;i<n;++i){
            for(int j=0;j<m;++j){
                p[i][j]=pre[i*m+j]*suf[i*m+j+1]%MOD;
            }
        }
        return p;
    }
};
```

- 时间复杂度: $O(NM)$
- 空间复杂度: $O(NM)$
