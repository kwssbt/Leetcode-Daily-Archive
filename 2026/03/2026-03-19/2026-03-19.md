# [3212. 统计 X 和 Y 频数相等的子矩阵数量](https://leetcode.cn/problems/count-submatrices-with-equal-frequency-of-x-and-y/)

## 前缀和

`'X'` 视为 1，`'Y'` 视为 -1，`'.'` 视为 0

维护前缀和

将【 X 和 Y 频数相等】转化为【矩阵和为 0 】

用或运算维护 `bool x[][]` 表示前缀是否含 `'X'`

`x[i+1][j+1]=x[i][j+1]||x[i+1][j]||(grid[i][j]=='X')`

如果把【包含`gird[0][0]`】的要求去掉

`'X'` 视为 1，`'Y'`，`'.'` 都视为 0

维护前缀中 `'X'` 的数量

可以 $O(1)$ 求任意子矩阵中 `'X'` 的数量

---

## 代码实现 (C++)

```cpp
class Solution {
public:
    int numberOfSubmatrices(vector<vector<char>>& grid) {
        int n=grid.size();
        int m=grid[0].size();
        vector pre(n+1,vector<int>(m+1));
        vector x(n+1,vector<int>(m+1));
        int ans=0;
        for(int i=0;i<n;++i){
            for(int j=0;j<m;++j){
                int val=0;
                if(grid[i][j]!='.'){
                    val=grid[i][j]=='X'?1:-1;
                }
                pre[i+1][j+1]=pre[i][j+1]+pre[i+1][j]-pre[i][j]+val;
                x[i+1][j+1]=x[i][j+1]+x[i+1][j]-x[i][j]+(grid[i][j]=='X');
                if(pre[i+1][j+1]==0&&x[i+1][j+1]>0){
                    ans++;
                }
            }
        }
        return ans;
    }
};
```

- 时间复杂度: $O(NM)$
- 空间复杂度: $O(NM)$
