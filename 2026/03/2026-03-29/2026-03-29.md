## 分奇偶

奇数位之间可以随意交换

偶数位之间可以随意交换

可以分别对奇数位，偶数位排序，比较是否一致，$O(N\ logN)$

或者分别对奇数位，偶数位的字符记数，检查是否一致，$O(N)$

---

## 代码实现 (C++)
```cpp
class Solution {
public:
    bool canBeEqual(string s1, string s2) {
        int n=s1.size();
        vector<int>odd1(26),eve1(26);
        vector<int>odd2(26),eve2(26);
        for(int i=0;i<n;++i){
            if(i&1){
                odd1[s1[i]-'a']++;
                odd2[s2[i]-'a']++;
            }
            else{
                eve1[s1[i]-'a']++;
                eve2[s2[i]-'a']++;
            }
        }
        for(int i=0;i<26;++i){
            if(odd1[i]!=odd2[i]||eve1[i]!=eve2[i]){
                return 0;
            }
        }
        return 1;
    }
};
```

- 时间复杂度: $O(N)$
- 空间复杂度: $O(1)$
