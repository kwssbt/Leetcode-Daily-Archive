# [2573. 找出对应 LCP 矩阵的字符串](https://leetcode.cn/problems/find-the-string-with-lcp/)

## 构造&DP

### 尝试构造

尝试构造一个长度为 `n` 的字符串 `s` ，要求字典序最小

从 `'a'` 开始填字符，`s[0]='a'`

并找到所有必须填 `'a'` 的位置，填上 `'a'`

然后填 `'b'` ，最靠左的空位填上 `'b'`

并找到所有必须填 `'b'` 的位置，填上 `'b'`

…………

直到全部填满。

### 检查

以上法构造的 `s` 只是有可能正确

比如对于样例 `[[4,3,2,1],[3,3,2,1],[2,2,2,1],[1,1,1,3]]`

我们只用了 `[4,3,2,1]` 就构造出了 `s = "aaaa"`，并没有用到 `[[3,3,2,1],[2,2,2,1],[1,1,1,3]]`

而 `s = "aaaa"` 真正的 `lcp` 应该是 `[[4,3,2,1],[3,3,2,1],[2,2,2,1],[1,1,1,1]]`

所以我们要检验一下我们所构造的 s 真正的 lcp 与 题目条件是否一致

### 构造 lcp （DP）

`lcp[i][j] 等于子字符串 s[i,...,n-1] 和 s[j,...,n-1] 之间的最长公共前缀的长度`

- 对于`s[i]!=s[j]`

    `s[i,...,n-1]`和`s[j,...,n-1]`没有公共前缀，`lcp[i][j] = 0` 

- 对于`s[i]==s[j]`

    `s[i,...,n-1]`和`s[j,...,n-1]`的公共前缀长度 = `s[i+1,...,n-1]`和`s[j+1,...,n-1]`的公共前缀长度 + 1

    `lcp[i][j] = lcp[i+1][j+1] + 1`

- 特别地，当`s[i]==s[j]`并且`i==n-1 || j==n-1` 时 `s[i][j]=1`

## 代码实现 (C++)
```cpp
class Solution {
    vector<vector<int>>LCP(string&s){
        int n=s.size();
        vector lcp(n,vector<int>(n));
        for(int i=n-1;i>=0;--i){
            for(int j=n-1;j>=0;--j){
                if(s[i]!=s[j]){
                    lcp[i][j]=0;
                }
                else{
                    if(i==n-1||j==n-1){
                        lcp[i][j]=1;
                    }
                    else{
                        lcp[i][j]=1+lcp[i+1][j+1];
                    }
                }
            }
        }
        return lcp;
    }
public:
    string findTheString(vector<vector<int>>& lcp) {
        int n=lcp.size();

        string s(n,'#');
        char ch='a';
        for(int i=0;i<n;){
            s[i]=ch++;
            for(int j=i+1;j<n;++j){
                if(lcp[i][j]){
                    s[j]=s[i];
                }
            }
            while(i<n&&s[i]!='#'){
                ++i;
            }
        }

        for(char c:s){
            if(c>'z'){
                return "";
            }
        }
        
        auto s_lcp=LCP(s);
        for(int i=0;i<n;++i){
            for(int j=0;j<n;++j){
                if(s_lcp[i][j]!=lcp[i][j]){
                    return "";
                }
            }
        }

        return s;
    }
};
```

- 时间复杂度: $O(N^2)$
- 空间复杂度: $O(N^2)$
