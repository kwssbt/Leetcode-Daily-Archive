# [3567. 子矩阵的最小绝对差](https://leetcode.cn/problems/minimum-absolute-difference-in-sliding-submatrix/)

## 枚举

枚举每一个 $k\times k$ 的子矩阵

---

## 代码实现 (C++)
```cpp
class Solution {
public:
    vector<vector<int>> minAbsDiff(vector<vector<int>>& grid, int k) {
        int n=grid.size();
        int m=grid[0].size();
        if(k==1){
            return vector<vector<int>>(n,vector<int>(m,0));
        }
        function<int(int,int)>f=[&](int x,int y)->int{
            vector<int>temp;
            for(int i=0;i<k;++i){
                for(int j=0;j<k;++j){
                    temp.push_back(grid[x+i][y+j]);
                }
            }
            ranges::sort(temp);
            int res=INT_MAX;
            for(int i=1;i<temp.size();++i){
                if(temp[i]!=temp[i-1]){
                    res=min(res,temp[i]-temp[i-1]);
                }
            }
            return res==INT_MAX?0:res;
        };
        vector<vector<int>>ans;
        for(int i=0;i+k<=n;++i){
            vector<int>row;
            for(int j=0;j+k<=m;++j){
                row.push_back(f(i,j));
            }
            ans.push_back(row);
        }
        return ans;
    }
};
```

- 时间复杂度: $O(NM k^2\log k^2)$
- 空间复杂度: $O(NM)$
