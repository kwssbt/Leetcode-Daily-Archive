## 循环数组

一个长为 $n$ 的循环数组

下标为 $i$ 的元素向右移动 $k$ 位（$k<n$），来到 $(i+k) \% n$

下标为 $i$ 的元素向左移动 $k$ 位（$k<n$），来到 $(i-k+n) \% n$


---

## 代码实现 (C++)
```cpp
class Solution {
public:
    bool areSimilar(vector<vector<int>>& mat, int k) {
        int n=mat.size();
        int m=mat[0].size();
        k%=m;
        for(int i=0;i<n;++i){
            for(int j=0;j<m;++j){
                if(i&1){
                    if(mat[i][j]!=mat[i][(j+k)%m])return 0;
                }
                else{
                    if(mat[i][j]!=mat[i][(j-k+m)%m])return 0;
                }
            }
        }

        return 1;
    }
};
```

- 时间复杂度: $O(NM)$
- 空间复杂度: $O(1)$
