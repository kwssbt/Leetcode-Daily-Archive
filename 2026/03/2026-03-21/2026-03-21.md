## 代码实现 (C++)
```cpp
class Solution {
public:
    vector<vector<int>> reverseSubmatrix(vector<vector<int>>& grid, int x, int y, int k) {
        int x1=x,x2=x+k-1;
        while(x1<x2){
            for(int j=y;j<y+k;++j){
                swap(grid[x1][j],grid[x2][j]);
            }
            x1++;
            x2--;
        }
        return grid;
    }
};
```

- 时间复杂度: $O(k^2)$
- 空间复杂度: $O(1)$
