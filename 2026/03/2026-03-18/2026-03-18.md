## 代码实现 (C++)
```cpp
class Solution {
public:
    int countSubmatrices(vector<vector<int>>& grid, int k) {
        int n=grid.size();
        int m=grid[0].size();
        vector pre(n+1,vector<int>(m+1));
        int ans=0;
        for(int i=0;i<n;++i){
            for(int j=0;j<m;++j){
                pre[i+1][j+1]=pre[i+1][j]+pre[i][j+1]-pre[i][j]+grid[i][j];
                if(pre[i+1][j+1]<=k){
                    ans++;
                }
            }
        }
        return ans;
    }
};
```

- 时间复杂度: $O(NM)$
- 空间复杂度: $O(NM)$
