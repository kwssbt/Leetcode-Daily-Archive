## 二分

对每个 `nums1[i]` 

在 `nums2` 的 [i,nums2.size()] 中找到最后一个比 `nums1[i]` 大的 `nums2[j]`

---

## 代码实现 (C++)
```cpp
class Solution {
public:
    int maxDistance(vector<int>& nums1, vector<int>& nums2) {
        int n=nums1.size(),m=nums2.size();
        int ans=0;
        for(int i=0;i<n;++i){
            int l=i,r=m-1,mid,j=i;
            while(l<=r){
                mid=(l+r)>>1;
                if(nums2[mid]>=nums1[i])j=mid,l=mid+1;
                else r=mid-1;
            }
            ans=max(ans,j-i);
        }
        return ans;
    }
};
```

- 时间复杂度: $O(N\log N)$
- 空间复杂度: $O(1)$
