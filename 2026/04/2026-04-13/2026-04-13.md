## 代码实现 (C++)
```cpp
class Solution {
public:
    int getMinDistance(vector<int>& nums, int target, int start) {
        int ans=INT_MAX;
        for(int i=0;i<nums.size();++i){
            if(nums[i]==target){
                ans=min(ans,abs(i-start));
            }
        }
        return ans;
    }
};
```

- 时间复杂度: $O(N)$
- 空间复杂度: $O(1)$
