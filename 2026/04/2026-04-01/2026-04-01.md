# [2751. 机器人碰撞](https://leetcode.cn/problems/robot-collisions/)

## 模拟

最后的答案要求按照初始编号排序，所以先记录编号

按真实位置从左到右排序

从左到右遍历每一个机器人

栈里存向右走的机器人

遇到向左走的机器人就发生 “碰撞”

向左走的机器人击败栈里的所有机器人后到达最左边，结算

遍历结束，栈里剩下的就是抵达最右边的机器人，结算

## 代码实现 (C++)
```cpp
class Solution {
    struct robot{
        int id;
        int p;
        int h;
        bool toR;
    };
public:
    vector<int> survivedRobotsHealths(vector<int>& positions, vector<int>& healths, string directions) {
        int n=positions.size();

        vector<robot>robots;
        for(int i=0;i<n;++i){
            robots.push_back({i,positions[i],healths[i],directions[i]=='R'});
        }
        sort(robots.begin(),robots.end(),[](const robot&a,const robot&b){return a.p<b.p;});

        vector<pair<int,int>>id_h;

        stack<robot>st;
        for(auto&[id,p,h,toR]:robots){
            if(toR){
                st.push({id,p,h,toR});
            }
            else{
                if(st.empty()){
                    id_h.push_back({id,h});
                }
                else{
                    while(st.size()&&h>0){
                        if(st.top().h<h){
                            h--;
                            st.pop();
                        }
                        else if(st.top().h==h){
                            h=0;
                            st.pop();
                        }
                        else{
                            h=0;
                            st.top().h--;
                        }
                    }
                    if(h>0){
                        id_h.push_back({id,h});
                    }
                }
            }
        }
        while(st.size()){
            id_h.push_back({st.top().id,st.top().h});
            st.pop();
        }

        sort(id_h.begin(),id_h.end());
        vector<int>ans;
        for(auto [id,h]:id_h){
            ans.push_back(h);
        }
        return ans;
    }
};
```

- 时间复杂度: $O(N\ logN)$
- 空间复杂度: $O(N)$
