# [1320. 二指输入的的最小距离](https://leetcode.cn/problems/minimum-distance-to-type-a-word-using-two-fingers/)

**2026-04-12**

**题面难度：Hard**

## 记忆化搜索

`dis(u,v)` 从字母 u 到字母 v 的距离

`dfs(i,p1,p2)` 完成 `[i,n)` 且第一根手指在 p1 第二根手指在 p2 所需的代价

设置哑节点（手指的最初状态，不在任何节点上）到任意节点距离为 0

---

## 代码实现 (C++)
```cpp
class Solution {
public:
    int minimumDistance(string word) {
        function<int(int,int)>dis=[](int u,int v)->int{
            if(u==26){
                return 0;
            }
            return abs(u/6-v/6)+abs(u%6-v%6);
        };

        int n=word.size();
        vector mem(n,vector<vector<int>>(27,vector<int>(27,-1)));
        
        function<int(int,int,int)>dfs=[&](int i,int p1,int p2)->int{
            if(i==n){
                return 0;
            }
            if(mem[i][p1][p2]!=-1){
                return mem[i][p1][p2];
            }
            int tar=word[i]-'A';
            return mem[i][p1][p2]=min(
                dfs(i+1,p1,tar)+dis(p2,tar),
                dfs(i+1,tar,p2)+dis(p1,tar));
        };

        return dfs(0,26,26);
    }
};
```

- 时间复杂度: $O(N)$
- 空间复杂度: $O(N)$
