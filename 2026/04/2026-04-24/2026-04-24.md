## 代码实现 (C++)
```cpp
class Solution {
public:
    int furthestDistanceFromOrigin(string moves) {
        int l=0,_=0,r=0;
        for(char c:moves){
            if(c=='L'){
                l++;
            }
            else if(c=='R'){
                r++;
            }
            else{
                _++;
            }
        }
        return max(l,r)+_-min(l,r);
    }
};
```

- 时间复杂度: $O(N)$
- 空间复杂度: $O(1)$
