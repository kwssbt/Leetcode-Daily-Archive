# [3418. 机器人可以获得的最大金币数](https://leetcode.cn/problems/maximum-amount-of-money-robot-can-earn/)

**2026-04-02**

**题面难度：Medium**

## 动态规划

`dp[i][j][k]`: 在 `[i,j]` 处使用了 `k` 次机会的最大金币数

---

## 代码实现 (C++)
```cpp
class Solution {
public:
    int maximumAmount(vector<vector<int>>& coins) {
        int n=coins.size();
        int m=coins[0].size();

        vector dp(n+1,vector<vector<int>>(m+1,vector<int>(3,-0x3f3f3f3f)));
        for(int i=1;i<=n;++i){
            for(int j=1;j<=m;++j){

                int val=coins[i-1][j-1];
                
                if(i==1&&j==1){
                    if(val<0){
                        dp[i][j][0]=val;
                        dp[i][j][1]=0;
                        dp[i][j][2]=0;
                    }
                    else{
                        dp[i][j][0]=val;
                        dp[i][j][1]=val;
                        dp[i][j][2]=val;
                    }                 
                }
                else{
                    if(val<0){
                        dp[i][j][0]=max(dp[i-1][j][0],dp[i][j-1][0])+val;
                        dp[i][j][1]=max({dp[i-1][j][0],dp[i][j-1][0],dp[i-1][j][1]+val,dp[i][j-1][1]+val});
                        dp[i][j][2]=max({dp[i-1][j][1],dp[i][j-1][1],dp[i-1][j][2]+val,dp[i][j-1][2]+val});
                    }
                    else{
                        dp[i][j][0]=max(dp[i-1][j][0],dp[i][j-1][0])+val;
                        dp[i][j][1]=max(dp[i-1][j][1],dp[i][j-1][1])+val;
                        dp[i][j][2]=max(dp[i-1][j][2],dp[i][j-1][2])+val;
                    } 
                }
            }
        }
        return dp[n][m][2];
    }
};
```

- 时间复杂度: $O(NM)$
- 空间复杂度: $O(NM)$
