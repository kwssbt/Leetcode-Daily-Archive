## 代码实现 (C++)
```cpp
class Solution {
public:
    int maxDistance(vector<int>& colors) {
        int n=colors.size();
        int ans=-1;
        for(int i=0;i<n;++i){
            for(int j=i+1;j<n;++j){
                if(colors[i]!=colors[j]){
                    ans=max(ans,j-i);
                }
            }
        }
        return ans;
    }
};
```

- 时间复杂度: $O(N^2)$
- 空间复杂度: $O(1)$
