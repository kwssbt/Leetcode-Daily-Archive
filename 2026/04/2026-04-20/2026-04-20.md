## 暴力

数据量小，直接遍历

---

## 代码实现 (C++)
```cpp
class Solution {
public:
    int maxDistance(vector<int>& colors) {
        int n=colors.size();
        int ans=-1;
        for(int i=0;i<n;++i){
            for(int j=i+1;j<n;++j){
                if(colors[i]!=colors[j]){
                    ans=max(ans,j-i);
                }
            }
        }
        return ans;
    }
};
```

- 时间复杂度: $O(N^2)$
- 空间复杂度: $O(1)$

## 贪心

首尾不同，直接返回 n-1

首尾相同

从左到右，找到第一个与尾不同的元素

从右到左，找到第一个与首不同的元素

## 代码实现 (C++)
```cpp
class Solution {
public:
    int maxDistance(vector<int>& colors) {
        int n=colors.size();
        if(colors[0]!=colors[n-1]){
            return n-1;
        }
        int l=0,r=n-1;
        while(colors[l]==colors[n-1])l++;
        while(colors[0]==colors[r])r--;
        return max(n-1-l,r);
    }
};
```

- 时间复杂度: $O(N)$
- 空间复杂度: $O(1)$
