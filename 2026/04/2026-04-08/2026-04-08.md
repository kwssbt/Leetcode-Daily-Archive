# [3653. 区间乘法查询后的异或 I](https://leetcode.cn/problems/xor-after-range-multiplication-queries-i/)

## 商分

分情况

**若k>=2**

直接处理的复杂度是 $O(log)$，可以接受

**若k=1**

直接处理的复杂度是 $O(N)$，无法接受

使用商分，复杂度 $O(1)$

商分数组 `d[l]` 乘上 `v`

`d[r+1]` 除以 `v` ，即乘上 `v` 的逆元

最后累乘

---

## 代码实现 (C++)
```cpp
class Solution {
    typedef long long ll;
    static const int MOD=1e9+7;
    ll qpow(ll b,ll e){
        ll res=1;
        while(e){
            if(e&1)res=res*b%MOD;
            b=b*b%MOD;
            e>>=1;
        }
        return res;
    }
    ll inv(ll x){
        return qpow(x,MOD-2);
    }
public:
    int xorAfterQueries(vector<int>& nums, vector<vector<int>>& queries) {
        int n=nums.size();
        
        vector<ll>d(n+1,1);
        for(auto&q:queries){
            int l=q[0],r=q[1],k=q[2];
            ll v=q[3];
            if(k==1){
                d[l]=d[l]*v%MOD;
                d[r+1]=d[r+1]*inv(v)%MOD;
            }
            else{
                for(int i=l;i<=r;i+=k){
                    nums[i]=nums[i]*v%MOD;
                }
            }
        }

        for(int i=0;i<n;++i){
            nums[i]=nums[i]*d[i]%MOD;
            d[i+1]=d[i+1]*d[i]%MOD;
        }

        int ans=0;
        for(int x:nums)ans^=x;
        return ans;
    }
};
```

- 时间复杂度: $O(Q\log N)$
- 空间复杂度: $O(N)$
