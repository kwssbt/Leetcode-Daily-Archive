# [874. 模拟行走机器人](https://leetcode.cn/problems/walking-robot-simulation/)

## 哈希&模拟

`-3 * 1e4 <= xi, yi <= 3 * 1e4` 用 `x * 1e5 + y` 作为哈希值，$O(1)$查找

`1 <= x <= 9` 很小，可以一步一步走，判断下一步是不是障碍物

---

## 代码实现 (C++)
```cpp
class Solution {
    long id(int x,int y){
        return (long)x*1e5+y;
    }
public:
    int robotSim(vector<int>& commands, vector<vector<int>>& obstacles) {
        unordered_set<long>obs;
        for(auto&ob:obstacles){
            obs.insert(id(ob[0],ob[1]));
        }
        int ans=0;
        int x=0,y=0;
        int mode=0;
        for(int op:commands){
            if(op==-2){
                mode=(mode-1+4)%4;
            }
            else if(op==-1){
                mode=(mode+1)%4;
            }
            else{
                if(mode==0){
                    for(int i=0;i<op;++i){
                        if(obs.count(id(x,y+1))){
                            break;
                        }
                        y++;
                    }
                }
                else if(mode==1){
                    for(int i=0;i<op;++i){
                        if(obs.count(id(x+1,y))){
                            break;
                        }
                        x++;
                    }
                }
                else if(mode==2){
                    for(int i=0;i<op;++i){
                        if(obs.count(id(x,y-1))){
                            break;
                        }
                        y--;
                    }
                }
                else{
                    for(int i=0;i<op;++i){
                        if(obs.count(id(x-1,y))){
                            break;
                        }
                        x--;
                    }
                }
                ans=max(ans,x*x+y*y);
            }
        }
        return ans;
    }
};
```

- 时间复杂度: $O(N+M)$
- 空间复杂度: $O(M)$
