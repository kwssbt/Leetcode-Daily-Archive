## 代码实现 (C++)
```cpp
class Solution {
public:
    bool judgeCircle(string moves) {
        int x=0,y=0;
        for(char c:moves){
            if(c=='R'){
                x++;
            }
            if(c=='L'){
                x--;
            }
            if(c=='U'){
                y++;
            }
            if(c=='D'){
                y--;
            }
        }
        return x==0&&y==0;
    }
};
```

- 时间复杂度: $O(N)$
- 空间复杂度: $O(1)$
