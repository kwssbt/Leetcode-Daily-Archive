# [3661. 可以被机器人摧毁的最大墙壁数目](https://leetcode.cn/problems/maximum-walls-destroyed-by-robots/)

**2026-04-03**

**题面难度：Hard** 

[查看PDF题面](./2026-04-03.png)

---

## DP&二分&排序

设计状态 

`dp[i][0]` 考虑 `[0,i]` 的机器人，并且第 `i` 个机器人向左的最大值（计算第 `i` 个机器人的向左的贡献）

`dp[i][0]` 考虑 `[0,i]` 的机器人，并且第 `i` 个机器人向右的最大值（不计算第`i`个机器人的向右的贡献）

先找到所有与机器人重合的墙，之后 dp 只考虑相邻两个机器人之间的墙

`ll` 第 `i-1` 个机器人向左，第 `i` 个机器人向左，它们之间的墙只能被第 `i` 个机器人打破

`rr` 第 `i-1` 个机器人向右，第 `i` 个机器人向右，它们之间的墙只能被第 `i-1` 个机器人打破

`lr` 第 `i-1` 个机器人向左，第 `i` 个机器人向右，它们之间的墙无法被打破

`rl` 第 `i-1` 个机器人向右，第 `i` 个机器人向左，它们之间的墙可以从两端被打破，不超过区间总墙数

最后得到 

`dp[n-1][0]` 考虑所有机器人，最后一个机器人向左

`dp[n-1][1]` 考虑所有机器人，最后一个机器人向右，需加上最后一个机器人向右打破的墙

## 代码实现 (C++)
```cpp
class Solution {
public:
    int maxWalls(vector<int>& robots, vector<int>& distance, vector<int>& walls) {
        int n=robots.size(),m=walls.size();

        vector<pair<int,int>>p_dis;
        for(int i=0;i<n;++i){
            p_dis.push_back({robots[i],distance[i]});
        }
        sort(p_dis.begin(),p_dis.end());

        int ans=0;
        sort(robots.begin(),robots.end());
        for(int wall:walls){
            int l=0,r=n-1,mid,f=0;
            while(l<=r){
                mid=(l+r)>>1;
                if(robots[mid]==wall){
                    f=1;
                    break;
                }
                else if(robots[mid]>wall){
                    r=mid-1;
                }
                else{
                    l=mid+1;
                }
            }
            if(f)ans++;
        }

        sort(walls.begin(),walls.end());
        function<int(int,int)>cnt=[&](int l,int r)->int{
            if(l>r)return 0;
            return upper_bound(walls.begin(),walls.end(),r)-lower_bound(walls.begin(),walls.end(),l);
        };

        vector dp(n,vector<int>(2,0));
        for(int i=0;i<n;++i){
            if(i==0){
                auto [p,dis]=p_dis[i];
                dp[i][0]=cnt(p-dis,p-1);
                dp[i][1]=0;
            }
            else{
                auto [pre_p,pre_dis]=p_dis[i-1];
                auto [cur_p,cur_dis]=p_dis[i];
                int ll=cnt(max(pre_p+1,cur_p-cur_dis),cur_p-1);
                int lr=0;
                int rl=pre_p+pre_dis<cur_p-cur_dis?cnt(pre_p+1,pre_p+pre_dis)+cnt(cur_p-cur_dis,cur_p-1):cnt(pre_p+1,cur_p-1);
                int rr=cnt(pre_p+1,min(pre_p+pre_dis,cur_p-1));
                dp[i][0]=max(dp[i-1][0]+ll,dp[i-1][1]+rl);
                dp[i][1]=max(dp[i-1][0]+lr,dp[i-1][1]+rr);
            }
        }

        return ans+max(dp[n-1][0],dp[n-1][1]+cnt(p_dis[n-1].first+1,p_dis[n-1].first+p_dis[n-1].second));
    }
};
```

- 时间复杂度: $O(N\log N + M\log M)$
- 空间复杂度: $O(N)$
