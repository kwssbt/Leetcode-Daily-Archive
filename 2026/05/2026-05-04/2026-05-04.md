## 矩阵翻转

先沿主对角线翻转，再沿中线翻转

---

## 代码实现 (C++)
```cpp
class Solution {
public:
    void rotate(vector<vector<int>>& matrix) {
        int n=matrix.size();
        for(int i=0;i<n;++i){
            for(int j=i+1;j<n;++j){
                swap(matrix[i][j],matrix[j][i]);
            }
        }
        for(int i=0;i<n;++i){
            int l=0,r=n-1;
            while(l<r){
                swap(matrix[i][l++],matrix[i][r--]);
            }
        }
    }
};
```

- 时间复杂度: $O(N^2)$
- 空间复杂度: $O(1)$
