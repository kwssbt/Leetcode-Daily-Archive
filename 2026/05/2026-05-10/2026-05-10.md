## 代码实现 (C++)
```cpp
class Solution {
public:
    int maximumJumps(vector<int>& nums, int target) {
        int n=nums.size();
        vector<int>dp(n,-1);
        dp[0]=0;
        for(int i=0;i<n;++i){
            for(int j=i+1;j<n;++j){
                if(dp[i]!=-1&&abs(nums[i]-nums[j])<=target){
                    dp[j]=max(dp[j],dp[i]+1);
                }
            }
        }
        return dp[n-1];
    }
};
```

- 时间复杂度: $O(N^2)$
- 空间复杂度: $O(N)$
