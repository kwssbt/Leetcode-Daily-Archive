# [1914. 循环轮转矩阵](https://leetcode.cn/problems/cyclically-rotating-a-grid/)

## 暴力模拟

层数：min(n,m)/2

当前层左上角坐标（x,y） 右下角坐标（x1,y1）

当前层格数：t = (x1-x)* 2 + (y1-y)* 2

每次旋转一格，旋转 t 次

---

## 代码实现 (C++)
```cpp
class Solution {
public:
    vector<vector<int>> rotateGrid(vector<vector<int>>& grid, int k) {
        int n=grid.size(),m=grid[0].size();
        int level=min(n,m)/2;
        for(int i=0;i<level;++i){
            int x=i,y=i;
            int x1=n-1-i,y1=m-1-i;
            int t=k%((x1-x)*2+(y1-y)*2);
            while(t--){
                int temp=grid[x][y];
                for(int j=y;j<y1;++j){
                    grid[x][j]=grid[x][j+1];
                }
                for(int j=x;j<x1;++j){
                    grid[j][y1]=grid[j+1][y1];
                }
                for(int j=y1;j>y;--j){
                    grid[x1][j]=grid[x1][j-1];
                }
                for(int j=x1;j>x;--j){
                    grid[j][x]=grid[j-1][x];
                }
                grid[x+1][y]=temp;
            }
        }
        return grid;
    }
};
```

- 时间复杂度: $O(min(n,m)*NM*(N+M))$
- 空间复杂度: $O(1)$
