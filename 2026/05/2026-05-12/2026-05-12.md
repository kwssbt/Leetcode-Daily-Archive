## 代码实现 (C++)
```cpp
class Solution {
public:
    int minimumEffort(vector<vector<int>>& tasks) {
        sort(tasks.begin(),tasks.end(),[](auto&a,auto&b){return a[0]-a[1]>b[0]-b[1];});
        int ans=0;
        for(auto&t:tasks){
            ans=max(ans+t[0],t[1]);
        }
        return ans;
    }
};
```

- 时间复杂度: $O(N\log N)$
- 空间复杂度: $O(1)$
