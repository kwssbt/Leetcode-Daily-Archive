## 动态规划

向右只能跳到更小的格子，向左只能跳到更大的格子

维护前缀最大值`preMax`，后缀最小值`sufMin`

对 `ans[n-1]` 一定可以跳到 `[0,n-1]` 的最大值，也就是 `preMax[n-1]`

同理 对`ans[i]` 一定可以跳到 `[0,i]` 的最大值，也就是 `preMax[i]`

对于 [1,2,3,10,20,30] 

[1,2,3] 不能跳到 [10,20,30] 

因为 `preMax[2]` < `sufMin[3]`

从右向左处理

若 `preMax[i]` <= `sufMin[i+1]` : 跳不到右边，`ans[i]=preMax[i]`

若 `preMax[i]` > `sufMin[i+1]` : 可以从 i 跳到 i+1 ，`ans[i]=ans[i+1]`

---

## 代码实现 (C++)
```cpp
class Solution {
public:
    vector<int> maxValue(vector<int>& nums) {
       int n=nums.size();
       vector<int>preMax(n);
       preMax[0]=nums[0];
       for(int i=1;i<n;++i){
        preMax[i]=max(preMax[i-1],nums[i]);
       }
       vector<int>sufMin(n);
       sufMin[n-1]=nums[n-1];
       for(int i=n-2;i>=0;--i){
        sufMin[i]=min(sufMin[i+1],nums[i]);
       }
       vector<int>ans(n);
       ans[n-1]=preMax[n-1];
       for(int i=n-2;i>=0;--i){
        if(preMax[i]<=sufMin[i+1]){
            ans[i]=preMax[i];
        }
        else{
            ans[i]=ans[i+1];
        }
       }
       return ans;
    }
};
```

- 时间复杂度: $O(N)$
- 空间复杂度: $O(N)$
