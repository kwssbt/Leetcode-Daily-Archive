## 代码实现 (C++)
```cpp
class Solution {
public:
    int maxRotateFunction(vector<int>& nums) {
        int n=nums.size();
        int sum=0;
        int f=0;
        for(int i=0;i<n;++i){
            sum+=nums[i];
            f+=i*nums[i];
        }
        int ans=f;
        for(int i=n-1;i>=0;--i){
            f+=sum-n*nums[i];
            ans=max(ans,f);
        }
        return ans;
    }
};
```

- 时间复杂度: $O(N)$
- 空间复杂度: $O(1)$
