## 代码实现 (C++)
```cpp
class Solution {
public:
    vector<int> separateDigits(vector<int>& nums) {
        vector<int>ans;
        for(int x:nums){
            vector<int>t;
            while(x){
                t.push_back(x%10);
                x/=10;
            }
            for(int i=t.size()-1;i>=0;--i){
                ans.push_back(t[i]);
            }
        }
        return ans;
    }
};
```

- 时间复杂度: $O(NM)$
- 空间复杂度: $O(M)$
