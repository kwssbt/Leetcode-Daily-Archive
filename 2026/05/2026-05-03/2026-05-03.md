## 代码实现 (C++)
```cpp
class Solution {
public:
    bool rotateString(string s, string goal) {
        if(s.size()!=goal.size()){
            return 0;
        }
        for(int st=0;st<s.size();++st){
            int i=0;
            for(i;i<goal.size();++i){
                int j=(st+i)%s.size();
                if(s[j]!=goal[i]){
                    break;
                }
            }
            if(i==goal.size()){
                return 1;
            }
        }
        return 0;
    }
};
```

- 时间复杂度: $O(N^2)$
- 空间复杂度: $O()$
